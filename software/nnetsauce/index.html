About | Back

nnetsauce is a Python package for Statistical/Machine learning. In nnetsauce, pattern recognition is achieved by combining layers of neural networks.

These building blocks constitute the basis of many custom models, including models with deeper learning architectures; for regression, classification, and multivariate time series forecasting.

Here, are informations on package installation. Examples of use of nnetsauce can be found below.

Install | Back

Currently, nnetsauce can be installed through Github (but it will be available on PyPi in a few weeks). Here is how: 

          
            
            git clone https://github.com/thierrymoudiki/nnetsauce.git
            cd nnetsauce
            python setup.py install

Examples | Back

Here, are two examples of use of nnetsauce. A classification example on breast cancer data, and an illustrative example of deep learning. scikit-learn models are widely used in these examples, but nnetsauce will work with any learning model possessing methods fit() and predict() such as xgboost, LightGBM, or CatBoost.

It is also worth mentioning that sklearn's cross-validation functions, such as GridSearchCV and cross_val_score can be used on nnetsauce models.



classification example

We start by fitting a logistic regression to breast cancer data: 

          
            
            # Packages ----- 

            import nnetsauce as ns
            from sklearn import datasets, linear_model
            from sklearn.model_selection import train_test_split
            

            # Datasets -----

            breast_cancer = datasets.load_breast_cancer()
            Z = breast_cancer.data
            t = breast_cancer.target

            
            # Data splitting -----            

            Z_train, Z_test, t_train, t_test = train_test_split(
                Z, t, test_size=0.2, random_state=42)


            # Logistic regression -----

            regr = linear_model.LogisticRegression()            
            # fitting model on training set
            regr.fit(Z_train, t_train)

            # predictive accuracy on test set
            regr.score(Z_test, t_test)

The logistic regression model is now augmented with ten additional features with nnetsauce. Doing that, increases the classifier's accuracy (accuracy is used here for simplicity): 

          
            
            # Custom ns - Logistic regression -----

            fit_obj = ns.CustomClassifier(
            obj=regr,
            n_hidden_features=10,
            direct_link=True,
            bias=True,
            nodes_sim="sobol",
            activation_name="relu")            
            fit_obj.fit(Z_train, t_train)

            # predictive accuracy on test set
            fit_obj.score(Z_test, t_test)

deep learning example

This, is an illustrative example of deep learning with nnetsauce. Many, more advanced things could be tried. In this example, predictive accuracy of the model increases as new layers are added to the stack. 

          

            import nnetsauce as ns
            from sklearn import datasets, metrics

            diabetes = datasets.load_diabetes()
            X = diabetes.data 
            y = diabetes.target

            layer1_regr = linear_model.BayesianRidge()
            layer1_regr.fit(X[0:100,:], y[0:100])

            np.sqrt(metrics.mean_squared_error(y[100:125], layer1_regr.predict(X[100:125,:])))


            # layer 2 using layer 1 ----
            layer2_regr = ns.CustomRegressor(obj = layer1_regr, n_hidden_features=3, 
                                    direct_link=True, bias=True, 
                                    nodes_sim='sobol', activation_name='tanh', 
                                    n_clusters=2)
            layer2_regr.fit(X[0:100,:], y[0:100])


            # RMSE score
            np.sqrt(layer2_regr.score(X[100:125,:], y[100:125]))

            # layer 3 using layer 2 ----
            layer3_regr = ns.CustomRegressor(obj = layer2_regr, n_hidden_features=5, 
                        direct_link=True, bias=True, 
                        nodes_sim='hammersley', activation_name='sigmoid', 
                        n_clusters=2)
            layer3_regr.fit(X[0:100,:], y[0:100])

            # RMSE score
            np.sqrt(layer3_regr.score(X[100:125,:], y[100:125]))
